# DicWord
Dicionario para word office

##Arquivo txt para dicionário 
O arquivo para dicionário neste repositório contém mais de 320.000 palavras do idioma português brasileiro.

A fonte principal é a lista de palavras do corretor ortográfico LibreOffice, obtida aqui:

https://cgit.freedesktop.org/libreoffice/dictionaries/plain/pt_BR/pt_BR.dic
